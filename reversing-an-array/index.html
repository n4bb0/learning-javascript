Reversing an array

Exercise from Eloquent JavaScript

write two functions, reverseArray and reverseArrayInPlace.
The first, reverseArray, takes an array as argument and produces a new array that has the same elements in the inverse order.
The second, reverseArrayInPlace modifies the array given as argument in order to reverse its elements.
Neither may use the standard reverse method.

My code:

  
      // return a new array which is the first one reversed
      function reverseArray(array) {
        let reversedArray = [];
        for (let i = 0; i < array.length; i++) {
          reversedArray.unshift(array[i]);
        }
        return reversedArray;
      }

      // return the same array reversed
      function reverseArrayInPlace(array) {
        for(let i = 0; i < array.length; i++) {
          let a = array.pop();
          array.splice(i, 0, a);
        }
        return array;
      }

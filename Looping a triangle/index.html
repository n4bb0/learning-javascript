Looping a triangle

Exercise from: Eloquent JavaScript

Write a loop that makes seven calls to console.log to output the following triangle (personal extra: make it visible on the page):

#
##
###
####
#####
######
#######

Exercise below

My code:


      var target = document.getElementById('exercise');
      var line = '';

      for (var i = 0; i < 7; i++) {
        line += '#';
        console.log(line);
        target.innerHTML += line + '<br>';
      };
